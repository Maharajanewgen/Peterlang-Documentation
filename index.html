Job Study in First proof

We will perform the job study once the file is transitioned to production for the initial proof pagination. The following points are regarded as showstopper issues, and we will raise a query if any of these points fail to meet the requirements. In such cases, we will cease from proceeding further.

  • General
    • Language details to be provided
    • eisbn details to be provided
    • Note type and numbering in note type to be matched
  • Running Head
    • While transitioning the initial proof to production, RH should be provided.
    • RH should not have end punctuation.
    • RH should match the book Design, i.e., the Contributor volume should include Author and title of contribution.
    • RH should not exceed 55 characters.
  • ToC
    • ToC Level should be matched with Login MS.
    • Content in the ToC should match with inside content.
    • TOC title should not be in English (Table of Contents) for non-english books.
    • TOC title should be "Contents" not "Table of Contents" for US titles

Table of Contents

For titles in the US Division, the Table of Contents (ToC) heading should be labeled as "Contents".

The ToC level must align with the original manuscript provided, and the level will be recorded accordingly.

Part titles should not include a folio number.

In the case of contributed volumes, a list of illustrations should not be included.

Running Head

In the case of a contributed volume, the running head should display the contributor's name on the left side and the contributed title on the right side.

Notes

In the case of contributed volumes, footnote numbering should begin anew for each chapter.

For single-author books, continuous footnotes should be maintained throughout the entire book.

Float Element

For Subsidy titles, the float element will be positioned according to the manuscript.

For Non-Subsidy titles, the float element will be situated at the top or bottom of the page.

For US and UK titles, the float element will be positioned at the top or bottom of the page.

Artwork

The art proof must be approved before the initial proof production.

Print and POD

We would like to request you to provide a separate print txt file for hardback titles to complete the print setup with Amazon KEP. The standard trim size used for CPI and other printers is not supported by Amazon KEP and I have listed below the Amazon KEP trim size for Hardback titles.

Interior trim (Millimeters) for Hardback titles:

  • 139.7 x 216 mm
  • 152.4 x 228.6 mm
  • 156 x 234 mm
  • 177.8 x 254 mm
  • 209.6 x 279.4 mm

Going forward, please resize the trim size for HB titles as below:

  • If the HB title Trim size is 14.8x21 cm then resize the file to 139.7 x 216 mm
  • If the HB title Trim size is 15.2x22.9 cm/15x22.5 cm then resize the file to 152.4 x 228.6 mm
  • If the HB title Trim size is 15.5x23 cm then resize the file to 156 x 234 mm
  • If the HB title Trim size is 17.8x25.4 then Amazon txt file trim size should be 177.8 x 254 mm

Please request all PM's to provide a separate print txt file for hardback titles as per this request.

For paperback titles, we will use the print txt file provided for other printers to set up the title with Amazon KEP as Custom trim size is supported by Amazon KEP for paperback titles.
