Job Study in First proof

We will perform the job study once the file is transitioned to production for the initial proof pagination. The following points are regarded as showstopper issues, and we will raise a query if any of these points fail to meet the requirements. In such cases, we will cease from proceeding further.

  • General
    • Language details to be provided
    • eisbn details to be provided
    • Note type and numbering in note type to be matched
  • Running Head
    • While transitioning the initial proof to production, RH should be provided.
    • RH should not have end punctuation.
    • RH should match the book Design, i.e., the Contributor volume should include Author and title of contribution.
    • RH should not exceed 55 characters.
  • ToC
    • ToC Level should be matched with Login MS.
    • Content in the ToC should match with inside content.
    • TOC title should not be in English (Table of Contents) for non-english books.
    • TOC title should be "Contents" not "Table of Contents" for US titles
